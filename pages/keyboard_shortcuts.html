Keyboard Shortcuts

Keyboard shortcuts are a great way to navigate your way around a computer. It can cut down on time and just make using a computer feel more useful. One of the key benefits of using keyboard shortcuts is not having to take your hands off the keyboard to use the mouse. My personal favorite thing about keyboard shortcuts are to quickly complete an action without having to reach for my mouse and for quick, accurate commands.
Some of the most common keyboard shortcute are: ctrl+p to print, ctrl+c to copy, ctrl+v to paste, and ctrl+x to cut and copy.
These are very useful, but there are so many more shortcuts that may be used to navigate the computer to make your life easier, faster and using a computer feel much more fun. To cover some of the shortcuts I utalize daily, I have listed them below with a summary of what the shortcut does.


Keyboard Shortcuts with ctrl

  • ctrl+E
    Prompt for a search in browser with default search engine
  • ctrl+L
    Propt for entering URL in browser
  • ctrl+W
    Closes the window in the active browser session
  • ctrl+T
    Opens a new tab
  • ctrl+shift+T
    Opens most recently closed tab - repeating reopens most recent chain of tab(s) closed

Keyboard Shortcuts with WinKey

  • WinKey+I
    Opens the OS/Windows System Settings
  • WinKey+P
    Opens menu to change display between multiple screens: PC Screen Only | Duplicate | Extend | Second Screen Only
  • WinKey+L
    Locks screen - Useful when needing to quickly step away from your computer
  • WinKey+E
    Opens File Explorer
  • WinKey+V
    Opens copied clipboard / Turn on if haven't been on before - This maintains previously copied content in a compiled list
  • WinKey+D
    Minimizes all opened windows showing Desktopy
  • WinKey+[number]
    Opens application in taskbar: 1 = first taskbar application, 2 = second taskbar application, etc...
  • WinKey+Tab
    Opens all applications in a preview to view everything opened

Conclusion

I hope whomever finds these shortcuts may find ways to incorporate them into their daily routine. There are MANY more, but these are just a few of my favorite and most used shortcuts.
